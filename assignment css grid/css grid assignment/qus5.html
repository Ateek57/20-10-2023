justify-items

justify-items is used to control the alignment of all items within a grid or flex container. It sets the alignment for all items within the container, and you apply it to the container element.

justify-self

justify-self is used to control the alignment of a specific item within a grid or flex container. It overrides the alignment set by justify-items for that specific item. You apply justify-self to individual items within the container
